# Plan Template

> Fuer konkrete Umsetzungsvorhaben. Ablage: `context/plans/PLAN-NNN-<beschreibung>.md`

```markdown
# PLAN-NNN: <Plan-Titel>

## Ziel
Was soll erreicht werden?

## Kontext
Warum ist das noetig? Welche Abhaengigkeiten gibt es?

## Scope
- **IN Scope**: ...
- **NICHT in Scope**: ...

## Schritte
1. Schritt 1
2. Schritt 2
3. ...

## Betroffene Dateien
- `pfad/zur/datei.ext` — Was wird geaendert

## Risiken
- Risiko 1 → Mitigation
- Risiko 2 → Mitigation

## Akzeptanzkriterien
- [ ] Kriterium 1
- [ ] Kriterium 2
```

## Namenskonvention

- Offene Plans: `PLAN-NNN-<beschreibung>.md` (dreistellig, fortlaufend)
- Abgeschlossene: `finished/PLAN-FNNN-<beschreibung>.md`
