# Plans

All plans in the Lab, grouped by their `status` frontmatter property.

The folder structure under `/plans/` is **flat** — there is no `active/` or `finished/` subfolder. Status is metadata that can change without renaming or moving anything. The lists below are filtered live from frontmatter at build time.
